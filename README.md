## ERP API Service

Laravel‑сервис, отвечающий за внешний и внутренний API‑слой ERP‑платформы.

### Основные задачи

- **REST API для интеграций** с внешними системами (CRM, телефония, биллинг и др.).
- **Аутентификация и авторизация** запросов, проверка прав доступа.
- **Работа с очередями** для тяжелых/фоновых задач.
- **Интеграция с основным ERP‑приложением** (`../main`) и сервисом воркеров (`../worker`).

### Технологии

- Laravel 11 (PHP 8.1+).
- Очереди (queue workers), уведомления, нотификации.
- Возможная интеграция с `php-ffmpeg`, Google 2FA и др. пакетами (см. `composer.json`).

### Запуск локально (базовый пример)

```bash
cp .env.example .env
composer install
php artisan key:generate
php artisan migrate
php artisan serve
```

После запуска API будет доступен по адресу, указанному в `APP_URL` (по умолчанию `http://localhost:8000`).
