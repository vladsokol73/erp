## ERP Worker Service

Laravel‑сервис для фоновой обработки задач и вспомогательных интерфейсов ERP‑системы.

### Основные задачи

- **Выполнение фоновых джоб** (очереди Laravel) для тяжёлых и долгих операций.
- **Обработка интеграционных событий** (webhooks, задачи из основного приложения `../main` и ERP API `../api`).
- **Отрисовка вспомогательных UI‑панелей и страниц**, связанных с фоновыми процессами.

### Технологии

- PHP 8.1+, Laravel 11.
- Очереди (Redis/другая очередь, в зависимости от настроек `.env`).
- Поддержка Tailwind/Blade‑шаблонов и статических ассетов (`public/assets`).

### Базовый запуск локально

```bash
cp .env.example .env
composer install
php artisan key:generate
php artisan migrate

php artisan serve                # HTTP‑часть
php artisan queue:work           # воркер очереди
```

Точные параметры соединений (БД, Redis и др.) задаются в `.env`.
